# Android Mp3player

## 주제

안드로이드 스마트폰 외장메모리에 저장된 음악을 가져와 목록에 추가하고
들을 수 있는 Mp3plyaer

## 실행하기전

외장메모리의 MediaStore.Audio.Media.GENRE값을 가져오지 못하여
음악마다 장르값을 설정해주었습니다

## 기능

#### 1.음악재생
   + 재생목록, 장르, 좋아요 화면 어디서든 음악을 클릭하면
   원하는 리스트에서 재생가능

#### 2.좋아요
   + 음악재생 화면에서 하트를 누르면 좋아요 체크가 되고
   좋아요 음악리스트에 추가
   
#### 3.카운트
   + 음악 재생버튼을 누른횟수 만큼 재생회수 증가

#### 4.이전곡, 다음곡
   + 이전or 다음 버튼을 누르면 이전곡 or 다음곡으로 넘어감
